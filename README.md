# MadhuSiri

MadhuSiri is an Android application for bee farmers and growers. It helps manage hives, send spray alerts, and protect pollinators with a clean Compose UI, Google Maps support, and Firebase integration.

## Features

- Hive map with location tracking
- Spray alert messaging for nearby beekeepers
- AI assistant and health dashboard
- Firebase-backed data storage
- Modern Jetpack Compose interface

## Setup

1. Clone this repository:

```bash
git clone https://github.com/rahaann/Madhu-siri.git
```

2. Open the project in Android Studio.
3. Sync Gradle.
4. Add `google-services.json` to `app/`.
5. Configure Google Maps and Firebase credentials as needed.

## Run

- Start an Android emulator using `Tools > AVD Manager`, or connect a device with USB debugging.
- Click the green ▶ Run button in Android Studio.

## Notes

- Do not commit sensitive API keys or `google-services.json` if they contain private credentials.
- `local.properties` is ignored by Git and should contain your Android SDK path.
